Versions

TcSS uses semantic versioning. All previous versions are available. Older CDNs are all production level.

  • Production level CDN:

    • No traffic limits or throttling. Files are served via CloudFlare's CDN.

    Files can be automatically optimized if added ?min=1 query parameter.

    Using a specific tag or commit hash in the URL (not a branch). Files are cached permanently based on the URL. Query string is ignored.
